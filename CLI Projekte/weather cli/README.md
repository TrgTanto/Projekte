# Wetter CLI Tool (Python)

Ein kleines Python-Konsolenprogramm zum Abrufen von Wetterdaten über die OpenWeatherMap API.

Das Projekt wurde erstellt, um meine Python-Kenntnisse zu verbessern und praktische Erfahrung mit APIs, Benutzerinteraktion und Fehlerbehandlung zu sammeln.

## Funktionen

- Wetterabfrage über Städtenamen
- Wetterabfrage über Breitengrad und Längengrad
- Ausgabe von:
  - Land
  - Wetterzustand
  - Temperatur
  - Mindest- und Höchsttemperatur
- Fehlerbehandlung für:
  - ungültige API-Schlüssel
  - überschrittenes API-Limit
  - falsche Benutzereingaben

## Verwendete Technologien

- Python 3
- requests
- OpenWeatherMap API
