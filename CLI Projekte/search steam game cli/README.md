# Steam Game Info Tool (Python)

Ein kleines Python-Konsolenprogramm zum Abrufen von Informationen über Steam-Spiele mithilfe der Steam Store API.

Das Projekt wurde erstellt, um meine Python-Kenntnisse zu verbessern und praktische Erfahrung mit APIs, JSON-Datenverarbeitung, Benutzerinteraktion und Fehlerbehandlung zu sammeln.

## Funktionen

- Suche nach Steam-Spielen über den Namen
- Ausgabe von:
  - Spielname
  - App-ID
  - Preis
  - Angebotspreis
  - unterstützte Betriebssysteme
  - Spiel-Icon URL
- Erkennung von kostenlosen Spielen
- Anzeige von aktuellen Angeboten
- Fehlerbehandlung für:
  - ungültige Spieleingaben
  - nicht gefundene Spiele

## Verwendete Technologien

- Python 3
- requests
- Steam Store API
