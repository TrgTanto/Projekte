# Passwort Generator (Python)

Ein moderner Passwort-Generator mit grafischer Benutzeroberfläche, entwickelt mit Python und CustomTkinter.

Die Anwendung ermöglicht das Erstellen sicherer Passwörter mit anpassbaren Optionen sowie die Berechnung verschiedener Hash-Werte.

Das Projekt wurde erstellt, um praktische Erfahrung mit GUI-Entwicklung, Sicherheitsthemen und Python-Anwendungsstruktur zu sammeln.

## Funktionen

### Passwort-Generator

- Generierung sicherer Passwörter
- Einstellbare Passwortlänge
- Auswahl verschiedener Zeichenarten:
  - Großbuchstaben
  - Kleinbuchstaben
  - Zahlen
  - Sonderzeichen
- Passwort-Stärkeanzeige
- Passwort direkt in die Zwischenablage kopieren
- Neues Passwort per Klick generieren

### Erweiterte Funktionen

Zusätzlicher Hash-Rechner mit Unterstützung für:

- MD5
- SHA-1
- SHA-224
- SHA-256
- SHA-384
- SHA-512
- SHA3-224
- SHA3-256
- SHA3-384
- SHA3-512
