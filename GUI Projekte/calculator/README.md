Ein moderner Taschenrechner mit grafischer Benutzeroberfläche, entwickelt mit Python und CustomTkinter.

Das Projekt wurde erstellt, um praktische Erfahrung mit GUI-Programmierung, Benutzerinteraktion und Python-Anwendungsentwicklung zu sammeln.

## Vorschau

Der Taschenrechner bietet:

- modernes Dark-Mode Design
- grundlegende Rechenoperationen
- Prozentrechnung
- dynamische Anpassung der Schriftgröße
- Fehlerbehandlung bei Division durch 0

## Funktionen

- Addition
- Subtraktion
- Multiplikation
- Division
- Prozentrechnung
- Löschen einzelner Zeichen
- Alles zurücksetzen (AC)
- automatische Anpassung der Schriftgröße bei langen Zahlen
- grafische Benutzeroberfläche mit CustomTkinter

## Verwendete Technologien

- Python 3
- CustomTkinter
- Tkinter
